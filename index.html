O MicroFramework Definitivo para APIs PHP

Construa APIs RESTful modernas, rápidas e seguras. O Parrot PHP integra FastRoute, Eloquent ORM e JWT em uma arquitetura limpa e testável.

Começar Agora Ver a Arquitetura
bash
$ git clone https://github.com/arthur-oliveira-oficial/parrot-php.git
$ cd parrot-php && composer install
$ php -S localhost:8000 -t public/

Tudo que você precisa, nada que você não precise.

Um ecossistema cuidadosamente selecionado para produtividade máxima.

Roteamento Ultrarrápido

Implementado com nikic/fast-route. Defina suas rotas via Dispatcher para um desempenho imbatível em APIs.

Eloquent ORM Integrado

O poder do banco de dados do Laravel de forma standalone. Models, Migrations e Seeders prontos no DatabaseCapsule.

Segurança por Padrão

Autenticação JWT completa (JwtAuthMiddleware), headers de segurança e prevenção contra ataques comuns de fábrica.

Fila de Middlewares

Implementação flexível baseada na PSR-15. Controle CORS, Rate Limiting e Tratamento de Erros via MiddlewareQueue.

Cobertura de Testes

Garantia de qualidade com PHPUnit configurado. Testes para Autenticação, Rotas e CRUD de usuários já implementados.

Injeção de Dependência

Gerenciamento de instâncias nativo e limpo no container.php. Desacople sua lógica com facilidade.

Padrões Limpos e Familiares

Se você já utilizou frameworks modernos de PHP, se sentirá em casa. O Parrot foca em manter as coisas simples através de uma estrutura lógica de Rotas, Controllers e Models.

  • Respostas padronizadas estendendo a classe Response.
  • Validação de Request com tipagem forte nativa do PHP.
  • Padrão Resource para formatação consistente das saídas de API. 
<?php

use App\Controllers\UserController;
use App\Middlewares\JwtAuthMiddleware;

// Grupo protegido por autenticação
$router->group(['middleware' => [JwtAuthMiddleware::class]], function ($router) {
    
    $router->get('/api/users', [UserController::class, 'index']);
    $router->post('/api/users', [UserController::class, 'store']);
    $router->get('/api/users/{id:\d+}', [UserController::class, 'show']);
    
});

<?php

namespace App\Controllers;

use App\Models\UserModel;
use App\Views\UserResource;
use App\Core\Request;
use App\Core\Response;

class UserController extends Controller
{
    public function index(Request $request): Response
    {
        $users = UserModel::all();
        return $this->json(UserResource::collection($users));
    }
}

<?php

namespace App\Models;

class UserModel extends EloquentModel
{
    protected $table = 'usuarios';

    protected $fillable = [
        'nome',
        'email',
        'senha',
        'role'
    ];

    protected $hidden = [
        'senha',
    ];
}
