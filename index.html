Pronto para o PHP 8.2+

Crie APIs RESTful poderosas e sem enrolação

O Parrot PHP é um MicroFramework limpo, moderno e rápido. Combina o melhor do FastRoute, Eloquent ORM e autenticação JWT sob os padrões PSR. Sem a sobrecarga de grandes frameworks.

Começar agora Documentação Completa

Requerimentos: PHP 8.2+, Composer, MySQL/PostgreSQL.

config/routes.php 
<?php

use FastRoute\RouteCollector;
use App\Controllers\UserController;
use App\Controllers\AuthController;

// Roteamento limpo e declarativo
return function(RouteCollector $r) {
    
    // Autenticação
    $r->addRoute('POST', '/api/login', [AuthController::class, 'login']);
    
    // Rotas protegidas (Requer JWT)
    $r->addGroup('/api/users', function (RouteCollector $r) {
        $r->addRoute('GET', '', [UserController::class, 'index']);
        $r->addRoute('POST', '', [UserController::class, 'store']);
        $r->addRoute('GET', '/{id:\d+}', [UserController::class, 'show']);
    });
};

Por que o Parrot PHP?

Tudo o que você precisa, nada além disso.

Desenvolvido para ser o ponto de partida ideal para microsserviços e APIs focadas em performance, mantendo a DX (Developer Experience) incrível.

Roteamento Ultrarrápido

Implementação baseada no nikic/fast-route, garantindo o processamento de rotas mais rápido disponível no ecossistema PHP.

Eloquent ORM

Aproveite a sintaxe elegante e poderosa do Eloquent (do Laravel) para interagir com seu banco de dados, independente do framework.

Segurança "Out-of-box"

Middlewares nativos de Autenticação JWT, Rate Limiting (evita abusos), CORS configurável e Security Headers de fábrica.

Padrões PSR (7 e 15)

Totalmente compatível com as interfaces HTTP do PHP-FIG (Nyholm/psr7). Facilita a portabilidade e os testes unitários da aplicação.

Pronto para Produção

Acompanha configurações otimizadas para Docker e Caddy Server, garantindo HTTPS automático e deploy simplificado.

TDD Orientado

Ambiente pré-configurado com PHPUnit. Já inclui testes de CRUD de usuários, autenticação JWT e tratamentos de exceção.

Arquitetura Limpa e Intuitiva

Diga adeus à "mágica" excessiva. No Parrot PHP, o fluxo da requisição é claro e fácil de seguir. Você tem total controle sobre seus Controllers, Middlewares e Injeção de Dependências.

  • Padrão MVC Simplificado: Controllers enxutos retornando Respostas padronizadas.
  • Container Pimple: Injeção de dependências leve e poderosa.
  • Tratamento de Exceções Global: Retornos em JSON automáticos para erros 404, 401, 500, etc.
